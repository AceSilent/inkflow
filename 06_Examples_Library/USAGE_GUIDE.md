# 📚 优秀文库使用指南

## 🎯 快速开始

### 1. 查看现有范文

```bash
# 查看所有分类
ls 06_Examples_Library/

# 查看某个分类的范文
ls 06_Examples_Library/dark_revenge/

# 阅读具体范文
cat 06_Examples_Library/dark_revenge/遮天_辰东_九龙拉棺.md
```

### 2. 添加新范文

**方法一：手动创建**

1. 进入对应分类目录
```bash
cd 06_Examples_Library/dark_revenge/
```

2. 复制模板文件，修改内容

**方法二：使用提取工具**

```bash
python tools/extract_example.py \
    --book "凡人修仙传" \
    --author "忘语" \
    --category "traditional_xianxia" \
    --scene "韩立初入修仙界" \
    --content "..." \
    --tags "仙侠 冷静 无修饰" \
    --techniques "动作干脆 心理克制"
```

### 3. 在代码中使用

```python
from src.utils.example_library import get_example_library

# 获取文库管理器
library = get_example_library()

# 按分类获取
example = library.get_by_category('dark_revenge', random_choice=True)

# 按书名获取
example = library.get_example('遮天', author='辰东')

# 搜索范文
results = library.search(
    category='comedy_funny',
    tags=['#对话自然', '#吐槽'],
    max_count=3
)
```

## 📝 范文质量标准

### 优秀范文标准

✅ **动作描写干脆**
```
好: "他拔剑，刺向敌人的咽喉。"
差: "他缓缓地拔出了那柄闪烁着寒光的剑，然后刺向了敌人的咽喉要害。"
```

✅ **对话自然**
```
好: "叶修说："退役了，不打了。""
差: ""我决定退役了。"叶修意味深长地说道，眼神中透露出一丝不舍。"
```

✅ **无AI高频词**
避免: "仿佛"、"似乎"、"宛如"、"如潮水般"、"眼底闪过一丝"、"嘴角勾起一抹"

✅ **节奏紧凑**
```
好: 短句为主，快速推进
差: 长句堆叠，节奏拖沓
```
