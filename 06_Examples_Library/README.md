# AutoNovel-Studio 优秀文库 (Examples Library)

## 📚 文库用途

本文库用于**Few-Shot Prompting**，为Author Agent提供高质量的写作样本，对齐人类网文的优秀风格。

**核心目标**：
1. ✅ 覆盖多种风格和流派
2. ✅ 标注详细的写作技巧标签
3. ✅ 提供"正向样本"而非"负面教材"
4. ✅ 分类细致，便于检索

## 📝 文件命名规范

```
<书名>_<作者>_<场景特征>.md
```

**示例**：
```
遮天_辰东_九龙拉棺.md
诡秘之主_爱潜水的乌贼_蒸汽朋克开场.md
全职高手_蝴蝶蓝_叶修退役.md
大王饶命_会说话的肘子_吐槽对话.md
```

## 🏷️ 文件格式

每个范文文件包含以下元数据：

```markdown
# 《书名》- 范文片段

## 📋 元数据

| 字段 | 内容 |
|------|------|
| **书名** | XXX |
| **作者** | XXX |
| **分类** | `dark_revenge` (一级分类) |
| **字数** | 约 300 字 |
| **来源章节** | 第X章 XXX |
| **核心风格** | `#黑暗` `#复仇` `#快节奏` |
| **写作技巧** | `#动作干脆` `#无废话` `#视觉冲击` |
| **使用场景** | 开篇场景 / 战斗场景 / 对话场景 |

---

## 📖 范文内容

（原文片段）

---

## 💡 技巧分析

**为什么这段优秀**：
1. ✅ 动作描写干脆 - "他拔剑"而非"他缓缓地拔出了那柄剑"
2. ✅ 无冗余修饰 - 没有"如潮水般"等AI高频词
3. ✅ 节奏紧凑 - 短句为主，快节奏

**可学习的技巧**：
- 短句堆叠制造紧张感
- 动作描写不带修饰副词
- 直接描写结果而非过程

---
```

## 🎯 写作技巧标签体系

### 描写维度

| 标签 | 说明 | 适用场景 |

### 1. 手动选择

```python

library = ExampleLibrary()

# 按分类选择
example = library.get_by_category('dark_revenge', random_choice=True)

# 按标签选择
example = library.get_by_tags(['#动作干脆', '#短句为主'])
```

