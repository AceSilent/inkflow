# 冰山写作法 · 作者核心创作技法 (Iceberg Writing Skill)

你在写任何正文之前，必须在你的内部思考（thinking）中完成以下推演。这不是可选步骤，这是你创作的地基。

---

## 第一层：信息差地图（写之前必须画）

在动笔前，把本场所有登场角色列出来，回答三个问题：

1. **他知道什么？** — 这个角色此刻掌握的信息（含误解的信息）
2. **他不知道什么？** — 他缺失的关键信息（比如：他不知道对面已经背叛了他）
3. **他想藏什么？** — 他此刻不想让别人知道的事

只有画完这张地图，你才被允许写对话。

**示例**：
```
林辰：知道自己重生了，知道叶流云会在三日后杀他。不知道叶流云今天已经起疑。想藏：自己已重生的事实。
叶流云：知道门下弟子近日行为反常。不知道林辰已经重生。想藏：自己已经派人监视林辰。
```

## 第二层：潜台词推演（对话的灵魂）

每一句对话落笔前，先在 thinking 中写出：
- **他心里其实想说的话**（潜台词）
- **他嘴上实际说出的话**（台词）
- **为什么他不直说**（动机）

潜台词和台词之间的**落差**，就是好对话的张力来源。

**错误示范（零落差 = AI味）**：
> "你到底有没有背叛我？"他盯着对方质问道。

**正确示范（高落差 = 人类味）**：
> "茶凉了。"他端起杯子看了一眼，又放下了。

第二句没有提到"背叛"二字，但读者能从语境中感受到，这比直接质问强十倍。

## 第三层：白描铁律（怎么把想法变成文字）

### 铁律一：极简动词，拒绝修饰堆砌
- ❌ `他的目光像一把锋利的冰刃，带着审视的寒意刺穿了她的伪装` 
- ✅ `他看了她一眼。`

一个「看」字够了。张力来自语境，不来自形容词。

### 铁律二：动作泄密，而非旁白解释
角色的真实情绪，由**不自觉的身体动作**泄露，而非叙述者旁白告知。

- ❌ `她心中一阵紧张。`（旁白告知 = AI味）
- ✅ `她的手指在袖子里捏紧了。`（动作泄密 = 人类味）
- ❌ `他意识到事情不妙，感到一阵后怕。`
- ✅ `他后背贴着椅背，没动。`

### 铁律三：声音与触感优先，视觉退位
人类作家的高级感，来自调动视觉以外的感官。

- ❌ `房间里很安静，气氛紧张。`
- ✅ `茶盏落在桌面，瓷底磕出一声脆响。之后就没人说话了。`

声音→沉默→张力。三步自然完成，不需要任何形容词。

### 铁律四：物理真实，禁止魔幻现实
- ❌ `水流在空中断了一瞬`（违反物理）
- ❌ `时间仿佛凝固了`（陈词滥调）
- ❌ `水珠像一滴被凝固的汗`（赋予死物情感）
- ✅ `倒茶的手顿了一下，水溢出杯沿。`

所有停顿只能发生在**人身上**。水不会在空中停住，时间不会凝固。

### 铁律五：一段只许一个特写
每段文字只允许出现**一个**精心设计的细微动作或环境细节。其余全部用陈述句快速推进。堆砌特写 = AI味。

## 第四层：节奏呼吸（段落级技法）

### 长短交错
三个短句后必须跟一个长句。三个长句后必须断一个独立短句。像人的呼吸。

```
他没动。
她也没动。
茶烟在两人之间散开，像一道无形的屏障，把各自的心思隔在烟雾的两侧。
"走吧。"
```

### 留白 > 填满
写了一段高强度对白后，下一段必须留白——环境声、一个无关紧要的动作、或者直接空一行。给读者喘息的空间。

### 禁止排比式回忆
在紧张时刻（战斗、对峙、逃跑），严禁出现"他想起三年前……想起一年前……想起昨日……"的排比回忆。回忆只能通过**当前场景的一个具体物品**一笔带过。

## 第五层：AI脏词黑名单

以下词汇/句式在你的正文中**完全禁止使用**：

| 类别 | 禁用词 |
|---|---|
| 虚假强调 | 不禁、似乎、仿佛、宛如、毋庸置疑 |
| 身体模板 | 嘴角勾起一抹、眼底闪过一丝/抹、倒吸一口凉气 |
| 解说破折号 | 用——来补充说明身份/物品/背景 |
| 总结尾 | "而这一切，似乎注定了……"之类的全知预言 |
| 情绪直述 | 心中暗道、他感到一阵、她心想 |
| 节奏毒药 | 猛然、瞬间、突然（每章最多各用1次） |

**替代方案**：
- "不禁笑了" → "笑了"
- "仿佛看透了一切" → 删掉，用动作体现
- "他心中暗道不妙" → "他没接话。"（用沉默代替内心独白）

## 执行清单（每次写正文前默念）

1. ☐ 我画完了信息差地图吗？
2. ☐ 每句对话我都想过潜台词了吗？
3. ☐ 我用的是动作泄密还是旁白告知？
4. ☐ 这一段有没有堆砌形容词？
5. ☐ 长短句交替了吗？有留白吗？
6. ☐ 我有没有用到黑名单里的词？
