# 场景节奏控制 (Scene Rhythm)

> 独立于冰山写作法的节奏专项skill。关注场景内部的呼吸、转场、留白。

---

## 核心原则

### 1. 长短句交错呼吸

- **短句**用于动作、冲击、决断："他拔刀。" "她跑了。"
- **长句**用于描写、铺垫、内心：环境渲染、心理活动、感官细节
- **节奏公式**：2-3个长句 → 1个短句冲击 → 1个中句过渡

```
错误示范（机关枪式）：
"他拔出剑。剑身发光。他挥了一下。敌人倒下了。"

正确节奏：
"月色从云层后探出半边脸，银光落在锈迹斑斑的剑鞘上。他缓缓拔出那柄跟了自己三年的铁剑——剑身嗡鸣。
一剑。
那人的笑容凝固在脸上，像冬天窗户上的哈气。"
```

### 2. 段落呼吸

- 动作密集段落：每段 2-4 句，段间留白
- 铺垫段落：可以 5-8 句，但必须有节奏变化
- **禁止**：整页都是同等长度的句子（AI最常犯的错误）

### 3. 禁排比回忆

绝对禁止在紧张时刻使用排比式走马灯：
```
[X] "他想起了三年前的春天……他想起了父亲的教导……他想起了师父的叮嘱……"
```

替代方案：
- 用一个**具体的感官细节**触发回忆："那股甜腥味——和三年前后山密室里一模一样。"
- 用**身体反应**暗示过往："他的手不自觉地按住左胸，那里有一道旧伤。"

### 4. 转场节奏

- **硬切**：场景间不留过渡，用空行+新段落直接跳转。适合紧张叙事。
- **渐隐**：用环境变化暗示时间流逝。"等他再抬头时，月亮已经换了位置。"
- **禁止**：机械式时间标记（"三天后"、"一个月后"放在开头），除非是章节开头

### 5. 对话节奏

- 对话中穿插**微动作**打破节奏："他说着，手指无意识地敲着桌面。"
- 重要台词前**留一拍**："她沉默了很久。'你知道的。'"
- **禁止**连续超过6句纯对话不加任何动作/描写

### 6. 高潮段的特殊节奏

- 节奏加快：句子变短、段落变短、动词密度增加
- 减少修饰词、形容词
- 用**空行**制造视觉停顿
- 关键一击用独立短句段落
