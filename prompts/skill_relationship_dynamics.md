# 角色关系网推演 (Relationship Dynamics)

> 角色之间的关系是活的网络，不是静态标签。每次互动都在改变网络的拓扑。

---

## 关系的三个维度

### 1. 信息差（Information Asymmetry）

角色之间知道不同的事情，这是叙事张力的核心来源：

- **A知B不知**：A可以操控、试探、隐藏
- **B知A不知**：造成戏剧性反讽（读者知道危险但角色不知道）
- **双方都不知**：第三方在暗中操控
- **双方都知但不说**：最深的张力——沉默比对话更有力

写对话时必须检查：**这两个角色此刻各自知道什么？** 信息差决定了对话的潜台词。

### 2. 权力关系（Power Dynamics）

- **明面权力**：师徒、上下级、强弱
- **暗面权力**：谁掌握更多信息、谁更冷静、谁有后手
- **权力翻转**：最有戏剧性的时刻。弱者突然占据主动，强者发现自己被算计

每段对话、每次互动都要问：**此刻谁占上风？这段结束后，权力天平移动了吗？**

### 3. 情感距离（Emotional Distance）

用数字量化（1-10，1=生死之交，10=陌生人/死敌）：

- 距离在缩短？通过什么——共患难？发现共同秘密？物理接触？
- 距离在扩大？通过什么——背叛？误会？价值观冲突？
- **每章至少一个关系的距离发生变化**，否则角色关系在原地踏步

## 关系推演规则

### 互动前检查

1. 两个角色上次见面是什么时候？期间发生了什么？
2. 他们各自对对方的最新认知是什么？（可能是错误的认知）
3. 这次互动，各自的**显性目标**和**隐性目标**分别是什么？

### 互动后更新

1. 信息差是否改变了？（有人暴露了秘密？有人获得了新情报？）
2. 权力关系是否移动了？
3. 情感距离是否改变了？往哪个方向？

### 关系网的蝴蝶效应

A和B的关系变化，会影响B和C的关系：
- 如果B发现A是叛徒，那B对A的盟友C的态度也会改变
- 如果A和B和好，那A和B各自的阵营关系也会调整
- 写到某个角色的关系变化时，检查是否有**关联角色**也需要更新

## 对话写作中的关系表现

- **亲密关系**：可以省略主语、用昵称、句子可以不完整（因为对方懂）
- **敌对关系**：措辞精确、每句话都在试探或攻击、表面客气实际杀机
- **上下级**：下级的措辞会更谨慎，语气词更多，会观察上级反应
- **伪装关系**：表面的措辞和内心的想法不一致——用动作/微表情暴露真意

## 避免的陷阱

- [X] 所有角色对主角都是同一种态度（要么全喜欢要么全讨厌）
- [X] 关系变化太突然（上一章还是敌人，下一章就成了朋友——没有过渡）
- [X] 重要关系只靠旁白交代（"他们是好朋友"——要用行动展示）
- [X] 忘记群戏中旁观者的反应（A和B在吵架，C在旁边不可能没反应）
